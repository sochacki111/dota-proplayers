# dota-proplayers

## Requirements
Docker installed on your machine

## Installation
Run following commands in project root directory:
```sh
docker-compose build
docker-compose up
```

## Technologies
✅ React + Redux
✅ TypeScript  
✅ Express.js  
✅ MongoDB  
✅ Passport.js (JWT authorization)  
✅ pino, morgan (logging)  
✅ Redis (request caching)  
✅ Docker
